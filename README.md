# 🧠 MRI Brain Tumor Detection Project

**Fast and efficient brain tumor detection using 2D and Pseudo-3D CNN models with Simple and Advanced preprocessing**



## Description

This project focuses on **automated detection of brain tumors from MRI images** using deep learning techniques.  
The main goal is to assist **researchers and medical professionals** in identifying tumors more quickly and accurately, reducing manual inspection workload.  

The approach leverages:

- **2D Convolutional Neural Networks (CNNs)** for classical image analysis  
- **Pseudo-3D CNN models** to capture richer feature representations  
- **Simple preprocessing** → image resizing + normalization  
- **Advanced preprocessing** → CLAHE contrast enhancement + Z-score normalization, which improves model performance  

This setup allows comparing **Simple vs Advanced preprocessing** and **2D vs 3D models** in terms of **accuracy, Dice coefficient, IoU score, and training speed**.


## Technologies & Libraries

- **Python 3.x**  
- **TensorFlow / Keras** – Building and training neural networks  
- **OpenCV** – Image preprocessing  
- **matplotlib** – Visualization of results  
- **scikit-image** – Additional image processing  
- **NumPy / scikit-learn** – Data handling and train/test splitting
